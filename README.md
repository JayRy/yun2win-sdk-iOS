# yun2win-sdk-iOS V_0.0.1

### 编译环境
编译环境：Xcode-Version 7.2.1 (7C1002)

-
### 目录结构

```
┌── API
│   ├── Common
│   │   ├── Config：全局配置
│   │   ├── Util：常用工具
│   │   ├── Category：扩展方法
│   ├── Yun2Win：SDK
│   ├── Model：SDK相关实体和方法
│   ├── APITests：单元测试
│   ├── Classes：Demo界面
│   │   ├── Login：登陆
│   │   ├── Setting：设置
│   │   ├── ConversationList：会话列表
│   │   ├── Conversation：会话
│   │   └── Contact：控制器，对应app中的各个页面
└── Pods 依赖管理工具
```
